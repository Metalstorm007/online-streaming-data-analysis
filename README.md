# Online Streaming Data Analysis

## Project Overview
This project performs exploratory data analysis on an online streaming platform dataset to understand content distribution, trends, and insights.

## Tools and Technologies
- Python
- Pandas
- Matplotlib
- Seaborn

## Objectives
- Analyze the distribution of Movies and TV Shows
- Explore content trends over the years
- Identify top countries producing streaming content
- Visualize data using graphs and charts

## Key Steps
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis
4. Data Visualization

## Libraries Used
- pandas
- matplotlib
- seaborn

## Project Files
- `netflix_analysis.py` : Python script for analysis
- `netflix_titles.csv` : Dataset used for analysis
